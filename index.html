Instructions:
Use W to jump when running.
Use W in mid-air to start flying.
While flying, use WASD to move around.
If you stop controlling the character, she will fall again to running stance after a while.
As this is a RPG, every flying, running and shooting parameter will be enhanced through skills and experience. Some of them can be already modified if you use Chrome Dev Tools/Firebug, just use the console and see the object GameManager.player.
